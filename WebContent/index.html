¿Cómo funciona?

Tuber es un sistema inteligente que mide el nivel de residuos de contenedores.

Arduino

Sistema Tuber utiliza una placa Arduino en conjunto con sensores ultrasónicos, programados para calcular la distancia entre el tope de un contenedor y los residuos contenidos en él. La información obtenida es enviada con la ayuda de un módulo Wi-Fi y el porcentaje de llenado del basurero es actualizado.

Aplicación Web

La aplicación utiliza la información de todos los contenedores almacenados y en base a ello es capaz de trazar, mediantes llamadas al API de Google Maps, el camino más corto y eficiente para que los camiones de basura puedan recolectar los residuos.
